Cloudflare Worker

Email Inbound Parse Webhook

本エンドポイントは、受信メールを解析し、Webhook 宛てに整形済みデータを転送するための Cloudflare Worker です。 ブラウザからの直接利用を目的とした公開アプリケーションではありません。

概要

メールヘッダおよび本文を抽出し、後続システムで処理しやすい形式へ変換して転送します。 設定が不完全な場合は早期に処理を停止し、不要な外部通信や追加処理を行わない設計です。

構成

  • 転送先 URL は WEBHOOK_URL として Cloudflare Secret で管理しています。
  • 設定未登録時はメールイベントを即時に reject し、重い処理へ進まないようにしています。
  • 本ページは静的アセットとして配信される案内用ページです。
